# ⚜️ Lily Beauty Salon

**Разработка премиального веб-ресурса для салона красоты в Германии.**

## 🎯 Суть проекта
Создание имиджевой витрины с последующей трансформацией в полноценную систему управления записями (CRM).

---

## 🛠 Технологический стек
*   **Backend:** Python (Django)
*   **Frontend:** HTML/CSS (Django Templates), Vanilla JS
*   **Database:** SQLite (Dev) → PostgreSQL (Prod)
*   **Integrations:** Telegram Bot API

---

## 📅 Этапы реализации

### 1. Витрина (MVP)
Имиджевый сайт. Главная задача — презентация бренда и доверие.
*   **Мастера:** Презентация топ-мастеров (Liliia Yakina).
*   **Контент:** Услуги, цены, портфолио работ.
*   **Trust-фактор:** Интеграция дипломов и сертификатов (карусель/грид).

### 2. Автоматизация (Future)
Внедрение системы онлайн-записи.
*   **Алгоритм "Тетрис времени":** Расчет свободных слотов в зависимости от длительности конкретной услуги.

### 3. Управление
Интеграция с Telegram-ботом.
*   Мгновенные уведомления персонала о новых записях.
*   Управление расписанием через мессенджер.
