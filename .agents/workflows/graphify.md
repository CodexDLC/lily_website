# Workflow: graphify
**Command:** /graphify
**Description:** Turn any folder of files into a navigable knowledge graph

## Steps
Follow the graphify skill installed at ~/.agents/skills/graphify/SKILL.md to run the full pipeline.

If no path argument is given, use `.` (current directory).
When working inside `lily_website`, prefer `graphify-out/wiki/index.md` if it exists.
For shared codex ecosystem questions, also consult `C:\install\projects\codex_tools\graphify-out\GRAPH_REPORT.md` and its `wiki\` folder if present.
