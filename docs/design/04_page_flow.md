# 04. СТРУКТУРА СТРАНИЦ И ЛОГИКА (PAGE FLOW)

[📂 Design Index](./README.md) > 📄 Page Flow

---

## 1. ГЛАВНАЯ СТРАНИЦА (HOME)

### Блок 1: Hero Split Section (Первый экран)
*   **Layout:** Две колонки (Split).
*   **Левая колонка (Текст):**
    *   Uptitle: "Premium Beauty Salon".
    *   Заголовок (H1): "Искусство Вашей Красоты" (с золотым акцентом).
    *   Описание: "Индивидуальный подход и немецкое качество...".
    *   **CTA Button:** "Записаться Онлайн" (Oval, Outline Gold).
*   **Правая колонка (Визуал):**
    *   PNG-изображение модели (без фона) с эффектом растворения внизу (Gradient Fade).
    *   *Особенность:* JS-ротатор маскотов (случайная модель при загрузке).

### Блок 2: Bento Grid (Услуги)
*   **Концепция:** Мозаика из карточек разного размера.
*   **Состав (6 карточек):**
    1.  **Парикмахерский зал** (Большая карточка, 2x2).
    2.  **Nail Service**.
    3.  **Косметология**.
    4.  **Brows & Lashes**.
    5.  **Массаж**.
    6.  **Депиляция**.
*   **Стиль:** Карточки с фоновыми изображениями и затемнением. При наведении — всплытие и золотая рамка.

---

## 2. СТРАНИЦА "КОМАНДА" (TEAM)

### Секция А: Владелец (Owner Block)
*   **Layout:** Карточка с фото слева и текстом справа.
*   **Контент:** Фото Лилии Якиной, приветственное слово, кнопка связи (Instagram).

### Секция Б: Топ-Мастера (Grid)
*   **Layout:** Сетка карточек (3 в ряд).
*   **Карточка мастера:**
    *   Фото (Zoom эффект при наведении).
    *   Имя + Специализация (напр. "Топ-Колорист").
    *   Кнопка "Профиль".

---

## 3. СТРАНИЦА "УСЛУГИ" (SERVICES)
*   **Структура:** Единый список всех категорий (Прайс-лист).
*   **Элемент:** "Парящий лист" (Floating Card) для каждой категории.
*   **Внутри категории:**
    *   Заголовок + Описание.
    *   Список топ-5 услуг (Название DE/UA, Цена, Время).
    *   Ссылка "Подробнее" (ведет на отдельную страницу категории, напр. `hair.html`).

---

## 4. ДЕТАЛЬНЫЕ СТРАНИЦЫ УСЛУГ (HAIR, NAILS, ETC.)
*   **Hero:** Фоновое изображение категории + Заголовок.
*   **SEO-текст:** Описание подхода и материалов.
*   **Полный Прайс-лист:** Все услуги категории.
*   **Мастера:** Мини-список мастеров, выполняющих эту услугу.
*   **CTA:** Кнопка записи внизу.
