## 2. СТРАНИЦА "КОМАНДА" (TEAM / MASTERS)

Реализуем жесткую иерархию: Личный бренд владельца + Экспертность команды.

### Секция А: Владелец (Owner Spotlight) — Layout "Side-by-Side"
Выделенный премиум-блок в начале страницы.
*   **Layout:** Горизонтальный.
    *   **Слева:** Крупный художественный портрет Лилии Якиной.
    *   **Справа:**
        *   Заголовок: Имя + "Founder & Art Director".
        *   Текст: Философия, регалии, опыт.
        *   Подпись: Рукописный шрифт (Script).
        *   **Кнопка:** "Записаться к топ-мастеру".

### Секция Б: Команда (Team Grid) — Layout "Square Cards"
Сюда попадают все остальные сотрудники.
*   **Группировка:** Разделение по специализации (Заголовок "Парикмахерский зал", затем "Nail-сервис").
*   **Вид карточки:** Строгий квадрат или вертикальный прямоугольник.
    *   Фото (статичное).
    *   Под фото: Имя + Грейд (напр. "Senior Master").
    *   *Hover-эффект (Наведение):* Появляется список топ-навыков (напр. "Блонд, Airtouch").
*   **Действие:** Клик по карточке открывает **Личное досье** (Модальное окно с дипломами и работами).

## 3. СТРАНИЦА "УСЛУГИ" (SERVICES)
*   **Навигация:** Табы сверху (Парикмахерский зал | Ногтевой сервис | Косметология).
*   **Список:** Горизонтальные карточки (Фото + Описание + Цена).
*   **Действие:** Кнопка "Записаться" в каждой строке.
*   **Логика:** При клике открывается форма, где выбрана конкретная услуга.
