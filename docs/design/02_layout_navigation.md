# 02. LAYOUT, GRID & NAVIGATION

## 1. Сетка и Отступы (Grid & Spacing)
Для создания ощущения премиальности критически важен «воздух». Контент не должен слипаться.

### Контейнер (Container)
*   **Desktop:** `max-width: 1200px` (или 1440px для широких экранов). Центрирование.
*   **Mobile:** Отступы по бокам `20px`.

### Вертикальный ритм (Spacing)
*   **Section Padding:** `80px - 120px` (Отступ между крупными блоками).
*   **Card Padding:** `24px - 32px` (Отступ внутри карточек).

### Z-Index (Слои)
*   **Level 0:** Основной фон.
*   **Level 1:** Изображение Хедера (Лого-баннер).
*   **Level 2:** Контент (Текст, Карточки).
*   **Level 3:** Навигация (Sticky Menu) и Модальные окна.

---

## 2. Header: Архитектура «Brand Overlay»
Шапка сайта объединена с главным баннером. Это единый визуальный блок.

### A. Фон (Background Layer)
Используется файл `logo_salon.jpg` (Фирменный зеленый паттерн с LILY).
*   **Главная страница:** Full-height Hero (высота 80-100vh).
*   **Внутренние страницы:** Compact Banner (высота 300-400px).
*   **Обработка:** Возможно легкое затемнение (Overlay) на 10-20%, чтобы белый/золотой текст читался контрастно.

### B. Инфо-бар (Top Bar) — Референс DLC
Располагается в самом верху, поверх фона.
*   **Позиция:** Выравнивание по правому краю (Right Aligned).
*   **Элементы:**
    *   Иконка телефона + Номер (+49 ...).
    *   Иконка конверта + Email (info@...).
*   **Стиль:** Тонкий шрифт, цвет Золото (`#E0C998`) или Белый. Без фона.

### C. Основная Навигация (Navbar)
Располагается под Инфо-баром (или по центру баннера — зависит от верстки).
*   **Лого:** Если фон перекрыт, дублируем текстовое лого "LILY" слева.
*   **Ссылки меню:** Главная | Услуги | Мастера | О нас.
*   **CTA Кнопка (Action):** "Записаться / Termin buchen".
    *   *Стиль:* Ghost Button (Прозрачный фон, золотая рамка 1px).

## 3. Footer: Концепция «Smart Drawer» (Умная шторка)
Вместо статичного огромного блока, который занимает место, футер работает динамически.

### Состояние А: «Элегантная полоска» (По умолчанию)
Это то, что видит пользователь, пока листает сайт.
*   **Вид:** Узкая, аккуратная линия в самом низу.
*   **Фон:** Deep Emerald (`#1A3C34`).
*   **Декор:** Тонкая золотая линия (`1px`) по верхней границе.
*   **Контент:** Минимум шума.
    *   Слева: Копирайт "© 2026 LILY Beauty Salon".
    *   Справа: Кнопка-текст "Контакты и Карта" с иконкой стрелки/плюса.

### Состояние Б: «Информационная панель» (Expanded)
«Массивный блок», который появляется при взаимодействии.
*   **Триггер (Как открыть):**
    *   Клик по полоске/стрелке.
    *   (Опционально) Автоматический плавный выезд (Slide Up), когда пользователь доскроллил страницу до самого конца.
*   **Контент (внутри развернутого блока):**
    *   Полноценная карта Google Maps (справа, как у DLC).
    *   Колонки с адресом, телефонами и ссылками на соцсети.
*   **Эффект:** Панель выезжает снизу вверх, перекрывая контент, или мягко раздвигает страницу.

---

## 4. Мобильная адаптация (Responsive)
*   **Header:** Инфо-бар скрывает Email, остается только кликабельная иконка Телефона.
*   **Menu:** Превращается в "Гамбургер" (три полоски). При клике выезжает меню сбоку (Offcanvas) на темно-зеленом фоне.
*   **Grid:** Карточки услуг (Топ-3) выстраиваются в одну колонку друг под другом.
