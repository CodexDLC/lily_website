# 🎨 Design (Визуал)

---

## 🎯 Описание
В этой папке хранятся материалы, касающиеся визуального стиля проекта: логотипы, цветовые палитры, шрифты, макеты и референсы.

## 🎨 Дизайн-код
*   **Стиль:** Классика, Премиум.
*   **Палитра:**
    *   🟢 **Глубокий изумрудный** (Deep Emerald)
    *   🟡 **Золото** (Gold)
    *   ⚪️ **Белый** (White)
*   **Типографика:** Акцент на шрифтах с засечками (Serif) для заголовков и чистых Sans-Serif для текста. Много "воздуха".

*   [**📄 01_Identity.md**](./01_identity.md) — Цветовая палитра, шрифты и общая концепция "Quiet Luxury".
*   [**📄 02_Layout_Navigation.md**](./02_layout_navigation.md) — Сетка, отступы, структура Хедера (Brand Overlay) и Футера (Smart Drawer).
*   [**📄 03_UI_Components.md**](./03_ui_components.md) — UI-Кит: Кнопки, Карточки услуг, Карточки мастеров и Формы.
*   [**📄 04_Page_Flow.md**](./04_page_flow.md) — Структура и логика страниц (Главная, Команда, Услуги).
*   [**📄 05_Content_Map.md**](./05_content_map.md) — Контент-стратегия, мультиязычность (i18n) и правила локализации.
*   [**📄 06_Booking_Flow.md**](./06_booking_flow.md) — Логика записи (Тетрис времени), сценарии пользователя и интеграция с Telegram.
