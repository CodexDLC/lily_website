## 1. Кнопки (Buttons)
Стиль кнопок должен быть легким, не перегружать интерфейс "кирпичами".

### Primary CTA (Основная кнопка)
Используется для действия "Записаться" в хедере и на обложках.
*   **Стиль:** Ghost Button / Outline (Призрачная кнопка).
*   **Фон:** Прозрачный.
*   **Рамка:** `1px solid #E0C998` (Sand Gold).
*   **Текст:** Uppercase (Заглавные), шрифт Sans-Serif, цвет Gold.
*   **Hover (Наведение):** Фон плавно заливается золотом (`#E0C998`), текст становится белым или темно-изумрудным.

### Secondary Link (Вторичная)
Для переходов "Подробнее о мастере", "Все услуги".
*   **Стиль:** Текст с подчеркиванием или стрелкой.
*   **Цвет:** Dark Grey (`#333333`).
*   **Декор:** Подчеркивание появляется только при наведении.

## 2. Карточки Услуг (Service Cards)
Используем два типа отображения для разных задач.

### Type A: "Magazine Teaser" (Для Главной)
Вертикальная карточка для блока "Популярное". Задача — продать картинкой.
*   **Пропорции:** Высокий прямоугольник (Portrait).
*   **Верх (60%):** Фото результата (на вылет, без полей).
*   **Низ (40%):** Белый фон.
    *   Название (Serif, крупно).
    *   Цена и Время (мелким шрифтом, серым).
    *   Кнопка-стрелка.

### Type B: "Price Row" (Для страницы Услуг)
Горизонтальная строка. Задача — удобный прайс-лист (Референс: DLC, но изящнее).
*   **Структура:**
    *   *Слева:* Фото-миниатюра (Квадрат или Круг).
    *   *Центр:* Название услуги + Описание (мелко).
    *   *Справа:* Цена (жирно) + Кнопка "Записаться" (Outline small).
*   **Разделитель:** Тонкая золотая линия между строками.

---

## 3. Карточка Мастера & Дипломы (Trust Unit)
*   **Фото:** Крупный портрет (ч/б или приглушенный цвет, при наведении — полноцвет).
*   **Инфо:** Имя (Serif) + Роль (Sans-Serif).
*   **Фича "Скрытые Дипломы":**
    *   Внизу карточки ссылка: `[+] Посмотреть квалификацию`.
    *   **Interaction:** При клике карточка "раздвигается" вниз (или открывается Pop-up), показывая аккуратную сетку (Grid) миниатюр сертификатов.

---

## 4. Формы (Inputs)
Для будущей записи.
*   **Стиль:** "Underline only" (Только нижняя линия).
*   **Фон:** Прозрачный.
*   **Бордер:** `1px solid #CCCCCC`. При фокусе — становится Золотым.
*   **Шрифт:** Крупный, легко читаемый.
