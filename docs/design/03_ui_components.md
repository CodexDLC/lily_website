## 1. Кнопки (Buttons)
Стиль кнопок должен быть легким, не перегружать интерфейс "кирпичами".

### Primary CTA (Основная кнопка)
Используется для действия "Записаться" в хедере и на обложках.
*   **Класс:** `.btn-pill`
*   **Стиль:** Ghost Button / Outline (Призрачная кнопка).
*   **Форма:** Oval (Скругление 50px).
*   **Фон:** Прозрачный.
*   **Рамка:** `1px solid var(--color-gold)`.
*   **Текст:** Uppercase (Заглавные), шрифт Sans-Serif, цвет Gold.
*   **Hover (Наведение):** Фон плавно заливается золотом, текст становится темным.

### Secondary Link (Вторичная)
Для переходов "Подробнее", "Профиль".
*   **Стиль:** Текст с подчеркиванием.
*   **Цвет:** Gold или White.
*   **Декор:** `border-bottom`.

## 2. Карточки Услуг (Service Cards)

### Type A: "Bento Card" (Для Главной)
Карточка для мозаики услуг.
*   **Фон:** Полноформатное изображение (Cover).
*   **Оверлей:** Градиент снизу вверх (от темного к прозрачному) для читаемости текста.
*   **Контент:**
    *   Заголовок (H3, Serif).
    *   Подпись (мелкий текст, описание).
    *   Стрелка (декор).
*   **Interaction:** При наведении карточка всплывает (`translateY`), рамка становится золотой.

### Type B: "Service Row" (Для страницы Услуг)
Строка в прайс-листе. Минимализм.
*   **Структура:**
    *   *Слева:* Название услуги (DE + UA/RU).
    *   *Справа:* Цена (Gold, Serif) + Время (Grey).
*   **Разделитель:** Пунктирная или тонкая линия между строками.
*   **Фото:** Отсутствуют в строках, чтобы не перегружать список.

---

## 3. Карточка Мастера (Team Unit)
*   **Контейнер:** "Парящая" карточка с темным фоном.
*   **Фото:** Крупный портрет, занимает 70-80% высоты.
    *   *Эффект:* Zoom фото при наведении.
*   **Инфо-блок:** Расположен под фото.
    *   Имя (Serif, крупно).
    *   Роль (Sans-Serif, мелко, uppercase).
    *   Кнопка "Профиль" (Secondary Link).

---

## 4. Формы (Inputs)
Используются на странице Контактов.
*   **Стиль:** "Underline only" (Только нижняя линия).
*   **Фон:** Прозрачный (`rgba(255,255,255, 0.02)` для контейнера).
*   **Бордер:** `1px solid rgba(255,255,255,0.3)`. При фокусе — Gold.
*   **Шрифт:** Белый, Sans-Serif.
